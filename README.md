# Personal Portfolio Website

A simple, professional portfolio website built using HTML and CSS and hosted with GitHub Pages.

## Features
- Responsive homepage layout
- Banner with background image
- Navigation menu
- Project showcase section
- Styled using modern CSS (hover effects, box shadow, border radius)
- Demonstrates CSS cascading (inline, internal, external)

## Technologies Used
- HTML5
- CSS3
- Git & GitHub

## Live Demo
https://Mukeshs-06.github.io/portfolio-website/

## Setup Instructions
1. Clone the repository
2. Open `example.html` in a browser
